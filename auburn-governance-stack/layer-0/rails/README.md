# Rails: A Governance Protocol for Artificial Intelligence

**Alpha Specification v1.0**

---

A protocol-layer architecture for AI governance — analogous
to TCP/IP for network communication. Establishes six core
governance primitives (Provenance, Custody, Verification,
Ledger, Redistribution, and Emergency) that enable
organizations to swap AI providers without rebuilding
governance infrastructure.

Includes the Five-Layer Architectural Stack (L0–L4), the
Six Core Rail Primitives, and the Interoperability Framework
for Outcome Equivalence.
