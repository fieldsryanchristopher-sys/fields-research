# The Model State Attestation Framework

**Evidence-Based Governance for Foundation Models**

---

Three-tier architecture composing hardware-rooted trust,
signed continuous internal invariant monitoring, and periodic
deep verification anchors. Survey of 200+ papers (2022–2026)
confirms no existing system composes these into a unified
attestation artifact.

Mapped to EU AI Act, FDA SaMD, SR 11-7, US Federal
Procurement, and AI Insurance. Theoretical ceilings documented
honestly: Rice's theorem, the Impossibility Sandwich, and TEE
side-channel limits.
