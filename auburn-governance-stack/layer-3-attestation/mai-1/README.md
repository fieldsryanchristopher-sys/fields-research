# The Model Attestation Interface (MAI-1)

**A Normative Profile and Conformance Protocol for
Foundation Model Governance**

---

The canonical IETF RATS-conformant attestation profile.
Composes three verification layers into a single verifiable
artifact. Specifies a CBOR/COSE-signed Entity Attestation
Token with five mandatory health invariants. Three-level
conformance test suite (MAI-C0/C1/C2) with binary pass/fail
semantics.

Mapped to seven regulatory frameworks, three CEN/CENELEC
harmonized standards, and specialist AI insurance requirements.
Copy-pasteable procurement clauses provided for federal RFPs,
EU conformity assessment, and enterprise vendor risk management.
