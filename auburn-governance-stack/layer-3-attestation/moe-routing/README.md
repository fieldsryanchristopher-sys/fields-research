# MoE Routing Attestation Specification

---

Normative specification for attesting Mixture-of-Experts
routing decisions. 19 requirements across three conformance
levels covering routing determinism, token dropping prohibition,
deployment topology disclosure, load balancing state attestation,
and expert utilization anomaly detection. Analyzes DeepSeek-V3,
Mixtral 8x7B, Switch Transformer, and ST-MoE. Extends MAI-1
with routing-specific fields.
