# CTS-1: MAI-1 Conformance Test Suite

**Binary Pass/Fail Enforcement for AI Attestation
Infrastructure**

---

45 derived test assertions across a four-level test pyramid.
Three conformance tiers (MAI-C0 research, MAI-C1 commercial,
MAI-C2 regulated/insured). 250+ vector negative test library.
Synthesizes conformance methodology from Common Criteria,
FIPS 140-3, OpenID Foundation, and IETF RATS.

Strictly binary pass/fail semantics. No partial compliance.
No grading curve. No interpretive discretion.
