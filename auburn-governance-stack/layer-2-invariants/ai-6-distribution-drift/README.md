# Clause AI-6: Distribution Drift Bound

---

The fourth mandatory MAI-1 invariant. Requires that KL
divergence between a model's production output distribution
and its cryptographically sealed baseline remain below a
certified threshold at every attested measurement point.
Foundation models are validated once and deployed indefinitely.
This specification provides the complete technical
infrastructure to operationalize continuous drift monitoring.
