## Three Reading Paths

**Path A: The Mathematician** — "I want to evaluate the proof."

Start with the main manuscript, the three Coq source files,
the FIELDS-COQ Deep Dive, the Permanent Axiom Trust
Boundary Verification, and the Subcritical Examinations.
After Path A, you have everything needed to find a
mathematical error if one exists.

**Path B: The AI Researcher** — "I want to understand the
governance methodology."

Start with this document (Lectern One), the Verification
Layer Architecture, Common Failure Modes Parts 1-5, the
FIELDS Deployment Guide, Auburn Model Reflections, and the
Ungoverned Audit Traces. 

**Path C: The General Reader** — "I want to understand what
happened."

Start with this document, the main manuscript Sections 1-4,
Common Failure Modes Part 5, and one Prompt Series document.
After Path C, you have the big picture.
