# Zero-Admits Analytic Infrastructure

**An Unconditional Bound for the Weighted Explicit Formula**

---

Earlier work. Part of the foundational research phase that
preceded the Auburn Governance Stack.

Certified analytic framework for the Riemann Hypothesis
adhering to Zero-Admits verification standards. Unconditional
bound for the prime-weighted sum using the classical de la
Vallée Poussin error term. Deterministic Noise Floor for prime
distribution variance. All constants explicitly bounded and
registered.
