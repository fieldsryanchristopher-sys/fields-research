# Parity Pattern Audit for the Collatz Map

**A Classification Framework with Empirical Descent Evidence**

---

Earlier work. Part of the foundational research phase that
preceded the Auburn Governance Stack.

Systematic audit of parity patterns under the Collatz map.
Lyapunov-like descent property across k-step windows.
Empirical demonstration of strictly negative drift for all
realized patterns across k in {6, 7, 8}. Includes public
verification script.
